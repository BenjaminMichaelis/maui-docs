# Visual Structure

The DataForm for .NET MAUI uses a set of visual elements when rendered.

## Displayed Elements

* `Header`
* `Editor`
* `Groups`
* `Validation for editor`
* `Error Message for editor`
* `Summer error message`&mdash;
